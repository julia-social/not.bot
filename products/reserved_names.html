Julia Vanity Name System

Think of reserved names as the "preview mode" for the upcoming Julia Vanity Name system. All of the reserved names will be pre-registered when that system launches. Pro subscribers get one reserved name, and Verified Creator subscribers get five. When the Julia Vanity Name system goes live, there will be no limit to the number of names you can buy.

The Julia Vanity Name system will be fully autonomous on the blockchain. The objectives of the Julia Vanity Name system are:

  • Anyone can register a name for an alias they own
  • No one can register a name for an alias they don't own
  • No one can prevent someone from registering a name
  • No one can forcibly take a name that has already been registered
  • Names do not expire
  • There will be a deposit required to register a name
  • The deposit for a name will be returned when the name is released
  • There will be a way to transfer or sell a name
  • The registry will not depend on ongoing maintenance from Julia Social

Presentation:

not.bot Reserved Names and Julia Vanity Names will have a ".nb" suffix. For example, "myname.nb". Default petnames have no suffix.

Verification:

None. If you want the vanity name "Tohichi_Hikita.nb" even though your name is not Tohichi Hikita, go for it! Also, don't assume that someone's vanity name has anything to do with who they are.

Time Delay:

It may take up to 10 minutes after registration for your reserved name to start showing up everywhere.

Allowed Characters

The reserved names system only allows the characters a-z, A-Z, 0-9, and underbar. The Julia Vanity Name system will have a larger set of available characters. Reserved names must be between 3 and 25 characters long. In considering whether a name has already been reserved, underbar is ignored, uppercase "I", lowercase "l" and digit "1" are considered the same, and uppercase "O" and digit "0" are considered the same. You cannot reserve names that could be confused for petnames.
